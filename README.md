# Algorithm Repository

A collection of C++ implementations of fundamental algorithms and lab exercises from CSE246 (Algorithm). This repository is intended for learning, practice, and quick reference for common algorithmic techniques.

## Topics Covered
 ### Searching Algorithms
 * Linear Search
 * Binary Search


### String Matching Algorithms

* Naive Algorithm
* KMP (Knuth–Morris–Pratt)

### Sorting Algorithms

* Insertion Sort
* Merge Sort
* Quick Sort

### Greedy Algorithms

* Activity Selection Problem
* Fractional Knapsack
* Job Sequencing with Deadlines
* Minimum Coin Change
* Huffman Coding

### Algorithm Design Techniques

* Greedy Method
* Divide and Conquer
* Optimization Strategies

### Graph Algorithm
* Kruskal’s Algorithm
